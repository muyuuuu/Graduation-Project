# Step 1

1. 安装软件；
2. 创建工程，包括原理图库、PCB库，原理图、PCB；
3. 元件库的创建，如电阻模型、排针类模型等；
4. 创建好的元件库模型的调用。

# Step 2

1. 放置元件，分割线分割模块(放置->绘图工具->线)；
2. 导线`(ctrl + w)`的添加和`NetLabel` 的添加；
3. 给原件赋值(工具->标注->原理图标注)；
4. 封装的统一管理(工具->封装管理器)；
5. 原理图的编译，常见错误的处理。

# Step 3

1. 绘制PCB元件库：焊盘，丝印，参数设置等；
2. 但不了解插针如何绘制，先准备往下看。
